## Artifacts/DDx — хранилище артефактов DDx-6D/8D

Эта папка предназначена для **сохранения результатов DDx-анализа в файловом виде** (индексы, аннотации, СМГ-реестр, матрицы, сценарии, заключение).

### Структура

- `Artifacts/DDx/_TEMPLATES/`: шаблоны файлов и “скелета кейса”.
- `Artifacts/DDx/<case_id>/`: папка конкретного кейса.

### `case_id`

- Формат: `YYYYMMDD_slug` (пример: `20260228_case01`).
- Только латиница, цифры, `_` и `-` (без пробелов).

### Масштабирование (chunking)

Для больших корпусов **не складывайте всё в один файл**. Используйте индексы и чанки:

- `02_annotations/index.md` + чанки `A-0001_A-0100.md`, `A-0101_A-0200.md`…
- `06_hcards/index.md` + чанки `H-0001_H-0050.md`, `H-0051_H-0100.md`…

### Что считается “минимумом”

См. `_TEMPLATES/README.md` и Skill `ddx-run-analysis`.
