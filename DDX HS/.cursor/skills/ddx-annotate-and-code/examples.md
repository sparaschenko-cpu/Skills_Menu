# Развернутые примеры аннотаций

## Пример 1: Фаза 1 -- дескриптивная аннотация

```
АНАЛИТИЧЕСКАЯ АННОТАЦИЯ

ID Аннотации: A-007
Источник: Транскрипт сессии 3 / стр.4 / строки 12-15

Исходный фрагмент текста:
> "Я опять не справился с этим проектом, как и всегда. Наверное, я просто не создан для такой работы."

БЛОК 1: КОНТЕКСТУАЛИЗАЦИЯ (DDx-6D)

- 1.1. Ось 1 (Интенция): 1.1.1 Сканирование
- 1.2. Ось 2 (Действие): 2.1.1 Атрибутивная грануляция
- 1.3. Ось 3 (Объект): А.1.1 Вербальные данные
- 1.4. Ось 4 (Масштаб): 2 Элементный
- 1.5. Ось 5 (Риск): 1 Дескриптивный
- 1.6. Ось 6 (Модус): Н/П

БЛОК 2: СОДЕРЖАНИЕ АНАЛИЗА

- 2.1. Наблюдение / Вывод: Высказывание содержит (a) самообесценивание ("не справился"), (b) генерализацию ("как и всегда"), (c) стабильную атрибуцию неудачи к неизменяемой внутренней причине ("не создан для").
- 2.2. Обоснование: Три элемента идентифицированы на лексико-семантическом уровне без интерпретации. Генерализация "как и всегда" = абсолютный квантификатор. "Не создан" = метафора врожденного дефицита.
- 2.3. Теоретическая линза: Н/П (Фаза 1, дескрипция)

БЛОК 3: СИСТЕМНАЯ ИНТЕГРАЦИЯ

- 3.1. Первичный код / Понятие: [самообесценивание], [генерализация неудачи], [стабильная внутренняя атрибуция]
- 3.2. Связь с гипотезами (ID): -- (гипотезы пока не сформированы)
- 3.3. Ключевые слова / Теги: #самооценка_негативная #атрибутивный_стиль #генерализация

БЛОК 4: МЕТА-АНАЛИЗ

- 4.1. Аналитическая рефлексия: Фрагмент вызывает легкое сочувствие. Отмечаю потенциальное желание "помочь" -- необходимо удерживать дескриптивную позицию. Альтернатива: "как и всегда" может быть риторическим усилением, а не реальной генерализацией.
```

## Пример 2: Фаза 2 -- интерпретативная аннотация

```
АНАЛИТИЧЕСКАЯ АННОТАЦИЯ

ID Аннотации: A-023
Источник: Транскрипт сессии 3 / стр.4 / строки 12-15

Исходный фрагмент текста:
> "Я опять не справился с этим проектом, как и всегда. Наверное, я просто не создан для такой работы."

БЛОК 1: КОНТЕКСТУАЛИЗАЦИЯ (DDx-6D)

- 1.1. Ось 1 (Интенция): 1.2.1 Идеация
- 1.2. Ось 2 (Действие): 3.2.1 Инверсия центрального конструкта
- 1.3. Ось 3 (Объект): B.1 Диагностическая гипотеза
- 1.4. Ось 4 (Масштаб): 3 Комплексный
- 1.5. Ось 5 (Риск): 3 Абдуктивный
- 1.6. Ось 6 (Модус): Н/П

БЛОК 2: СОДЕРЖАНИЕ АНАЛИЗА

- 2.1. Наблюдение / Вывод: Паттерн вербализации соответствует "пессимистическому атрибутивному стилю" (Seligman): внутренняя ("я"), стабильная ("не создан"), глобальная ("как и всегда") атрибуция негативных событий. Гипотеза: активная схема "дефективности/некомпетентности".
- 2.2. Обоснование: Три компонента пессимистического стиля (внутренняя + стабильная + глобальная) обнаружены в одном высказывании. При кросс-валидации с A-003, A-011 (аналогичные паттерны) -- устойчивый тренд.
- 2.3. Теоретическая линза: Когнитивно-поведенческая → Теория выученной беспомощности (Seligman) → Конструкт: пессимистический атрибутивный стиль

БЛОК 3: СИСТЕМНАЯ ИНТЕГРАЦИЯ

- 3.1. Первичный код / Понятие: [пессимистический атрибутивный стиль]
- 3.2. Связь с гипотезами (ID): Подтверждает H-03 (схема дефективности); Модифицирует H-01 (уточняет механизм)
- 3.3. Ключевые слова / Теги: #атрибутивный_стиль #выученная_беспомощность #схема_дефективности

БЛОК 4: МЕТА-АНАЛИЗ

- 4.1. Аналитическая рефлексия: Применена только когнитивная линза. Обязательство: создать альтернативную аннотацию через психодинамическую линзу (нарциссическая уязвимость?) для выполнения требования плюрализма. Вопрос: не является ли "не создан" защитным обесцениванием (вместо реального ощущения дефекта)?
```

## Пример 3: 8D-режим -- контекстуально-реляционная аннотация

```
АНАЛИТИЧЕСКАЯ АННОТАЦИЯ

ID Аннотации: A-041
Источник: Транскрипт сессии 5 / стр.2 / строки 8-12

Исходный фрагмент текста:
> "Когда начальник сказал, что отчет нужно переделать, я подумал -- ну вот, опять я всё испортил. Дома я бы так не переживал."

БЛОК 1: КОНТЕКСТУАЛИЗАЦИЯ (DDx-8D)

- 1.1. Ось 1 (Интенция): 1.1.2 Зондирование
- 1.2. Ось 2 (Действие): 2.1.3 Функциональная диссоциация
- 1.3. Ось 3 (Объект): A.2 Структурированные данные (нарративный блок)
- 1.4. Ось 4 (Масштаб): 3 Комплексный
- 1.5. Ось 5 (Риск): 2 Логико-структурный
- 1.6. Ось 6 (Модус): Н/П
- 1.7. Ось 7 (Ситуационный контекст): 7.1: Рабочий кабинет, обсуждение отчета с начальником / 7.2: Профессиональная сфера / 7.3: Корпоративная культура оценки результатов
- 1.8. Ось 8 (Реляционный контекст): 8.1: Диалог с внутренним критиком ("я всё испортил") / 8.2: Диада субъект-начальник (авторитетная фигура)

БЛОК 2: СОДЕРЖАНИЕ АНАЛИЗА

- 2.1. Наблюдение / Вывод: Самокритичная реакция активируется *специфически* в контексте оценки авторитетной фигурой (начальник). Субъект сам отмечает контекстуальную зависимость: "дома я бы так не переживал". Это ситуационно-специфичный, а не генерализованный паттерн.
- 2.2. Обоснование: Прямое указание субъекта на различие реакций в разных контекстах (работа vs дом) = свидетельство против чисто диспозиционной гипотезы.
- 2.3. Теоретическая линза: Н/П (структурное наблюдение)

БЛОК 3: СИСТЕМНАЯ ИНТЕГРАЦИЯ

- 3.1. Первичный код / Понятие: [ситуационная самокритика], [контекстуальная зависимость], [авторитетная фигура как триггер]
- 3.2. Связь с гипотезами (ID): Опровергает H-03 (если она сформулирована как генерализованная); Подтверждает H-05 (ситуационная тревога при оценке)
- 3.3. Ключевые слова / Теги: #ситуационная_зависимость #авторитет_триггер #контекстуальная_вариативность

БЛОК 4: МЕТА-АНАЛИЗ

- 4.1. Аналитическая рефлексия: Фрагмент является прямым аргументом для ситуационной атрибуции (H-Adaptive?). Необходимо: (1) проверить, есть ли другие контексты, где самокритика НЕ активируется (поиск исключений -- Режим 2); (2) запустить Сценарий C3 (диспозиция vs ситуация) перед тем, как принять H-03 как генерализованную.
```
